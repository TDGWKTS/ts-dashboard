# 轉運站數據儀表板
## Transfer Station Dashboard

實時儀表板用於 7 個轉運站，具有管理員功能。

### 功能特點
- 🔐 使用 SHA256 哈希的安全登錄
- 📊 實時數據可視化
- 📱 完全響應式設計
- 👥 基於角色的訪問控制
- 📈 交互式圖表和統計
- 📥 CSV 導出功能

### 轉運站
- IETS - 港島東轉運站
- IWTS - 港島西轉運站  
- NLTS - 北大嶼山轉運站
- NWNNTS - 西北新界轉運站
- OITF - 離島轉運設施
- STTS - 沙田轉運站
- WKTS - 西九龍轉運站 (管理員)

### 設置
1. 在 Google Apps Script 中更新您的 Sheet ID
2. 將前端部署到 Netlify
3. 在 WKTS 工作表中配置密碼

### 數據列
- 日期 (日期)
- 交收狀態 (狀態)
- 車輛任務 (任務類型)
- 入磅時間 (時間)
- 物料重量 (噸)
- 廢物類別 (分類)
- 來源 (地區)
